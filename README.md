
# QQBGM-163BGM-Lyrics_taskbar_display
可以让QQ音乐和网易云音乐的歌词显示才任务栏

## 启动参数


> **隐藏控制台窗口**

    ***.exe -noui

> **设置歌词采集频率**
> 实例每秒采集120次

    ***.exe --FPS 120
> **位置偏移**
> 实例X偏移10 Y偏移20

    ***.exe --X 10 --Y 20
> **位置偏移**
> 实例QQ音乐
> 0为QQ音乐 1为网易云音乐

    .exe --type 0
